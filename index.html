My Coding Projects

Space Wars

Our first schoolproject was to create a game of Space Wars. During this project, we learned to work with Python and how to work together as a development team.

Test Correct

Our second schoolproject was to create a website for the company Test-Correct. Test-Correct wants a platform that rates questions on a test on taxonomy. The rating needs to happen by an AI-model, after a JSON-file with the questions has been uploaded. During this project, we learned to work with html, css, sqlite, python and jinja.

Stichting Accessibility

Our third schoolproject was to create a platform for Stichting Accessibility. Stichting Accessibility wants a platform where organisations can request a research to rate their website on WCAG 2.1 criteria. During this project, we learned to work with Python, html, css, Javascript, SQLAlchemy, Jinja and Bootstrap.

FoodSave

During our fifth schoolproject, we worked on an application against foodwaste. This application allows the user to add and save groceries to a personal list. Afterwords the user gets notifications and meal suggestions for products that are about to expire. During this project we learned to work with React-Native, Typescript, Test-Driven-Development and SOLID-principles.

JSM Adviesbureau

For the company JSM Adviesbureau, I created a new website. This taught me to work with the React framework. It also taught me how to implement CMS, so the client can make adjustment to the website without having to adjust the code.
